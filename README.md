# Cisco ASA Site-to-Site VPN & Firewall Lab

A hands-on Cisco Packet Tracer lab simulating a site-to-site IPsec VPN
between two branch LANs over a WAN, with ASA firewall configuration.

## What's in this lab
- Cisco ASA site-to-site IPsec VPN connecting two branch LANs
- ASA firewall zones, ACLs, and static routes
- Crypto maps, ISAKMP policies, and pre-shared key authentication
- End-to-end ICMP connectivity testing between remote hosts

## Tools used
Cisco Packet Tracer, Cisco ASA Firewall

## Skills demonstrated
Network security, VPN configuration, IPsec, firewall policy,
ACLs, routing, ISAKMP/IKE, site-to-site tunneling
